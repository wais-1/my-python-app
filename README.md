## Особенности

- **Полноценный GUI** - современный интерфейс на PyQt6 с темной темой
- **CRUD операции** - создание, чтение, обновление, удаление записей
- **Четыре основные сущности**:
  - Производители антивирусов
  - Антивирусные продукты
  - Вредоносные программы
  - Сигнатуры обнаружения
- **Визуализация данных** - графики и диаграммы с использованием matplotlib
- **Экспорт в Excel** - полный отчет с аналитикой и визуализацией
- **Фильтрация и поиск** - расширенные возможности фильтрации по всем полям
- **Подробные карточки** - информация о каждой записи в удобном формате
